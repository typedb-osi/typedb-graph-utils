# typedb-graph-utils
Graph integrations for TypeDB. Please check the readme within the directory for the language you're interested in for a more detailed description.

Available languages:
* [python](python/README.md)
* [typescript](typescript/README.md)

Versioning and releases are per-language: see [`typescript/VERSION`](typescript/VERSION) and [`python/VERSION`](python/VERSION) (when added). Release procedures and CI conventions are documented in [`AGENTS.md`](AGENTS.md).
